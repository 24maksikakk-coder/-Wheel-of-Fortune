🎡 Wheel of Fortune – 
Увлекательная игра-викторина с вращающимся колесом, денежными призами, банкротством и бонусами!
Отгадывайте слова, крутите колесо, рискуйте и выигрывайте!
Реализована на 7 языках программирования с единым интерфейсом, цветной анимацией, режимами игры, ставками и таблицей рекордов.

🚀 Возможности
Вращающееся колесо – анимированный ASCII-спиннер с секторами разных номиналов.

Сектора колеса:

💰 Денежные призы (100–1000)

💎 Бонус (удвоение выигрыша)

💸 Банкротство (потеря всех очков за раунд)

🎁 Подарок (дополнительная попытка)

❓ Загадка (секретный приз)

Режимы игры:

Классический – угадайте слово по буквам, крутите колесо за право хода.

На выживание – при ошибке теряете все очки.

Турнир – несколько раундов с накоплением очков.

Подсказки – тема слова, количество букв, открытые буквы.

Ставки – рискните частью своего выигрыша для удвоения.

Статистика – количество игр, побед, общий выигрыш.

Рекорды – лучшие результаты сохраняются в файл.

Интерактивность – выбор букв, вращение колеса, принятие решений.

Цветной вывод – сектора, сообщения и результаты выделены.

📖 Использование
Синтаксис (единый для всех версий):

bash
<команда> [режим] [опции]
Режимы
classic – классическая игра (по умолчанию).

survival – режим выживания.

tournament – турнирный режим (3 раунда).

Опции
Опция	Описание
-w, --word <word>	Задать слово вручную (вместо случайного)
-t, --theme <theme>	Тема слова (например, животные)
-s, --stats	Показать статистику и выйти
-r, --reset	Сбросить рекорды
-h, --help	Справка
Управление
spin – вращать колесо.

letter <буква> – угадать букву (например, letter а).

guess <слово> – попытаться угадать слово целиком.

buy – купить гласную (стоимость 100 очков).

quit – выход.

Примеры
bash
# Классическая игра
python wheel.py classic

# Режим выживания с темой
python wheel.py survival -t животные

# Показать статистику
python wheel.py -s

🧠 Логика игры
Подготовка – выбирается случайное слово из словаря (или задаётся пользователем).

Колесо – содержит сектора с различными значениями, анимация вращения.

Ход игрока – вращает колесо, выпадает сектор:

Денежный – игрок может угадать букву, если она есть – получает сумму × количество букв.

Банкрот – теряет все очки за раунд.

Бонус – удваивает текущий выигрыш.

Подарок – даёт дополнительный ход.

Загадка – случайный приз (добавляет 50–300 очков).

Угадывание – игрок называет букву или слово целиком.

Победа – если слово отгадано, игрок получает накопленные очки.

Окончание – при ошибке или банкротстве ход переходит к компьютеру (или игре).

✨ Дополнительные фичи
Анимация вращения – показывается ASCII-спиннер с задержкой.

Покупка гласных – игрок может купить гласную за 100 очков.

Подсказка – отображается количество букв и уже открытые.

Сохранение рекордов – в ~/.wheel_record.json.

Цветной вывод – каждый сектор выделен своим цветом.
